# FinanceTrackerApp

## Description
A personal finance tracking application built with .NET MAUI and SQLite. 
Allows users to add, edit, and delete bills and payments.

## Features
- Add, edit, and delete bills
- Mark bills as paid/unpaid
- Persist data using SQLite

## Tech Stack
- .NET MAUI
- C#
- SQLite
